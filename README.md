# Makerspace Downloads

Public downloads for the Makerspace desktop app.

## Install the latest release

1. Go to the [latest release](https://github.com/anishthite/spanner-downloads/releases/latest).
2. Download the installer for your operating system:
   - **macOS:** download the `.dmg` file, open it, then drag the app into `Applications`.
   - **Windows:** download the `.exe` installer and run it.
   - **Linux:** download the provided Linux package, make it executable if needed, then run it.
3. Launch Makerspace after installation.

If your browser or operating system warns that the app was downloaded from the internet, confirm that you downloaded it from the GitHub release linked above before continuing.

## Older versions

Older installers are available on the [releases page](https://github.com/anishthite/spanner-downloads/releases).
